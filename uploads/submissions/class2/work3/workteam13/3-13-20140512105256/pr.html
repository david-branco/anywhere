O trabalho proposto a realizar consistia em criar um formulário em HTML que permitisse o envio dos sips para ser posteriormente tratados em PHP. Era suposto que o formulário permitisse a inserção de vários ficheiros, para isso foi usado JavaScript que assim permite mais facilmente adicionar ficheiros de uma forma dinâmica sem que seja necessário à partida definir um limite de ficheiros a enviar.

No abstract foi também inserido uma caixa de text editor que permite formatar o texto de uma forma mais intuitiva, feita também em JavaScript. Esta text editor permite seleccionar texto e pôr a negrito, sublinhado e itálico. Permite também criar texto sobre a forma de tópicos, nomeadamente li's, e também criar texto com numeração no início das linhas para melhor organização do abstract. A biblioteca usada para esta text editor chama-se ckEditor e tem milhares de possibilidades para criar a barra de ferramentas, o exemplo usado neste TP foi o mais minimalista possível porque a intenção era apenas mostrar as potencialidades que se pode adicionar uma text area em HTML. Esta biblioteca encapsula as tags HTML para os negritos, itálico, sublinhado e etc, o que facilita a inserção numa base de dados porque a informação segue em formato facilmente guardado, e permite também, ser posteriormente carregada mantendo todas as formatações inseridas pelo utilizador.
